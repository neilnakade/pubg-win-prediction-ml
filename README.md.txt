# PUBG Game Winner Prediction

## Project Objective
Predict the winning player based on in-game statistics using Machine Learning models.

## Models Implemented
- Logistic Regression
- Decision Tree
- Random Forest

## Model Comparison
Random Forest achieved the highest accuracy and is selected as the final production model.

## Future Improvements
- Hyperparameter tuning using GridSearchCV
- Cross-validation
- Advanced ensemble methods like XGBoost
